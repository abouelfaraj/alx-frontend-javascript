# 0x01. ES6 Promises

## Description

This project focuses on understanding ES6 Promises in JavaScript. It covers the concepts of Promises, async/await, try/catch, and other related topics. 

## Learning Objectives

By the end of this project, you should be able to:

- Explain the concept of Promises and their usage
- Understand how to use the `then`, `resolve`, and `catch` methods
- Utilize every method of the Promise object
- Implement try/catch for error handling
- Grasp the usage of the `await` operator and async functions

## Requirements

- All code files should use the `.js` extension.
- All code should be executed on Ubuntu 18.04 LTS using NodeJS 12.11.x.
- Use allowed editors: `vi`, `vim`, `emacs`, `Visual Studio Code`.
- All code files should end with a new line.
- A `README.md` file at the root of the project folder is mandatory.
- Your code should be tested using Jest with the command `npm run test`.
- Your code should be verified against lint using ESLint.
- All functions must be exported.
